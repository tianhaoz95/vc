# VC - Crypto Products Repository

A repository containing various products and utilities for cryptocurrency users.

- **[autopilot-dev](products/autopilot_dev/)**: An autonomous worker→reviewer coding-agent loop driven by a markdown task plan.
- **[Crypto.com → TurboTax Converter](products/crypto_tax/)**: A Flask web application to convert Crypto.com exports for tax reporting.
- **[Crypto.com → TurboTax Mobile (TBD)](products/crypto_tax_mobile/)**: Planned mobile companion.
- **[AI Developer Profile](products/ai_profile/)**: A serverless developer profile site with an on-device AI sidebar powered by WebLLM.
